# MIDA Skills Plugin

Coding skills for all MIDA workspace repos. Read the relevant skill before modifying any repo.

## Skills

- **mida-api**: CommonJS/Koa/Mongoose backend — routes, controllers, services, queues, jobs
- **mida-cms**: React/Redux/Polaris frontend — components, pages, hooks, services, state
- **mida-hm**: CommonJS/Koa/ClickHouse heatmap service
- **mida-proxy**: ESM/Koa reverse proxy — routing, auth, cluster selection
- **mida-recorder**: CommonJS/Koa session recorder — ping, quota, VIP sharding
- **mida-search**: CommonJS/Express/Elasticsearch — indexing, query, queue consumers
- **mida-mcp**: ESM/Express MCP server — tools, services, Redis caching
- **code-extension**: Vanilla JS/Webpack Shopify extension — modules, liquid, web pixel
